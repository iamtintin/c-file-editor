# File Editor in C

File Editor written in C that works using Command Line arguments. 

## Usage

There are plenty of comments within the file itself with reference to how to use the tool and what different features are available. 

### Help Page

A brief summary of the commands that can be used with the tool can be accessed by running the tool without any arguments.

```
gcc -o editor editor.c
./editor
```
